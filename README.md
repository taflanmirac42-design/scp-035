# SCP-035 Plugin

SCP: Secret Laboratory için SCP-035 (The Possessive Mask) eklentisi.

## Özellikler

- 🎭 Oyuncular maskayı takabilir
- ❤️ Başlangıçta 300 HP
- 📉 Zamanla HP azalması
- 👻 Başka bir cesede takılırsa vücut değiştirilir
- ✨ Görsel efektler

## Kurulum

1. Bu klasörü SCP:SL plugin klasörüne kopyala
2. Sunucuyu yeniden başlat
3. Plugin otomatik yüklenecek

## Kullanım

Oyun sırasında maskayı bulup takabilirsin. Maske seni kontrol edecek!
